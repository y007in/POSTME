## 주요 기능
- 하루 한 개의 지정된 질문에 답변 작성 가능
- 특정 날짜를 클릭 시 해당 날짜의 질문 및 답변 확인 가능
- 10개의 답변을 작성하면 마이페이지에 우체통이 생성되어 이전 답변들 확인 가능
